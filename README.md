# A22-Express

環境建置與需求：

Node Version Manager (nvm)
Node.js@14.16.0
Node Package Manager (npm)
Express@4.16.4

安裝方法：

1.於終端機輸入下列指令，或直接將程式碼打包下載
  git clone https://github.com/Ellierlin/A22-Express.git
2.進入專案資料夾
  cd A22-Express
下載所需套件
  npm install
執行專案，開啟虛擬本機伺服器
  npm run dev
於瀏覽器輸入網址
  http://localhost:3000/
